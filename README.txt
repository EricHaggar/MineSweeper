This archive contains the folders and files including all source codes of Assignment 2 completed in the ITI 1121/1521 course offered at the University of Ottawa.  In this Assignment, the Minesweeper game was created using the Model-View-Controller Method.  The bonus allows the user to right-click tiles to flag them as potential mines.

Note: to generate the Java Documentation, enter the following on the command line: javadoc -d doc .java -private
